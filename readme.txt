Wordpress WooCommerce to Google Sheet - Get orders directly to your google sheet.

== Description ==
Connecting your google sheet with your woocommerce site is just a click away. All orders received on WooCommerce Wordpress site will directly fetched in seconds to your google sheet. 

This plugin is completely free with all features included so there is no Pro version of this plugin.

== Installation ==
1. Upload `wp-woo-commerce-sync-for-g-sheet` to the `/wp-content/plugins/` directory
2. Activate the plugin through the \'Plugins\' menu in WordPress
3. Configure Google Api from the dashboard
